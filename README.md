# Дослідження методу Градієнтного Спуску (ГС) для розв'язання задачі мультилатерації (TDoA) методом Найменших Квадратів (МНК).

## Мета
1. Практично реалізувати та дослідити роботу алгоритму градієнтного спуску.
2. Зрозуміти суть Методу Найменших Квадратів (МНК) як способу формулювання функції втрат (loss function).
3. Візуалізувати та проаналізувати ключові параметри оптимізації: швидкість навчання (learning rate), вплив шуму (noise level) та початкове припущення (initial guess).
4. Дослідити інженерні трейдофи та фундаментальні проблеми реальних систем, такі як Геометричне Послаблення Точності (GDOP).

# Експериментальні Завдання

## 1. Контрольний Запуск ("Ідеальний світ")

### Налаштування:

- NOISE_LEVEL = 0 (повна відсутність шуму)
- LEARNING_RATE = 0.01
- INITIAL_GUESS = [50000, 50000]
- TRUE_POSITION = [45000, 35000]

### Аналіз та Контрольні Питання:

  Фінальна похибка позиціонування: 0.00 метрів. 
  Це означає, що після округлення до двох знаків результат збігся з реальною позицією. За ідеальних даних без шуму градієнтний спуск успішно знайшов точку мінімуму функції втрат.
  Може все ж бути невелика похибка через те, що алгоритм завершується не тоді, коли точка ідеальна, а коли зміна функції втрат між ітераціями стала меншою за поріг: delta_loss < 1e-7

Графік величини градієнта показує, як змінюється швидкість наближення алгоритму до оптимального розв’язку. На початкових ітераціях значення градієнта є великим, оскільки стартова точка знаходиться далеко від мінімуму функції втрат, тому алгоритм робить значні кроки та швидко змінює координати. У процесі наближення до правильної позиції градієнт поступово зменшується, що свідчить про зниження нахилу функції втрат і зменшення необхідності великих коригувань. Наприкінці роботи алгоритму значення градієнта наближається до нуля, тобто система досягає точки мінімуму, де подальші зміни координат майже не покращують результат. Це безпосередньо пов’язано з графіком Loss: на початку, коли градієнт великий, функція втрат зменшується швидко, а ближче до мінімуму, коли градієнт стає малим, зниження Loss сповільнюється.
